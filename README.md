# Ropegame

Ett en-gångs webbläsarspel inspirerat av Tangle Master/Rope Escape:
- 4 rep som sitter fast i en handled och flyttas mellan 5 punkter på en trästolpe
- 6 banor med tydligt högre svårighetsgrad
- Ingen tidspress, ingen inloggning, ingen sparning
- Mobilanpassat för iPhone och fungerar i webbläsare på Mac

## Starta
Öppna `index.html` i en webbläsare.

Tips: i VS Code kan du använda Live Server för enklare testning på både desktop och mobil.
